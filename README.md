# INTEROPen JSON Patient Summary Examples

Share examples of your IPS bundles as JSON in this repo!

Create a dir for your org. Add 1 or more JSON format IPS bundles so we all have access to examples from different providers and we can test importing each of them.

- **Send a PR** with JSON examples to the repo in a folder for your org!
- **Review** other IPS bundles and try out importing them to your system. 
- **Update** your examples as you tweak your output.
- **Interoperate!**

## Useful links

- IPS Viewer: https://ipsviewer.com/
- IPS Validator: https://ipsmern-dep.azurewebsites.net/schemavalidator - Click on **NHS SCR** in the top right.

## Hack Heavy Goals

- Build on the NHS-E IPS GitHub spec.
- Co-produce a draft UK Patient Summary Implementation Guide (UK-PS) based initially on EN requirements aligned to the global IPS IG 2.0
- Explore how we can safely and realistically support IPS-based sharing.
